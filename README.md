# Titan Programming Language

**Titan** is a new scripting langauge Built on top of **Rust**. 

Titan is a bit like a version of Python that was equipped with lower-level fetures and is compiled.

An Example:
```
typeln("Hello, World!")
x := 5
Num y = 6
typeln(Str(x + y))
```

Please note that **Titan**'s aim is to have ZERO dependncies.
